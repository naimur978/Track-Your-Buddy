## üì± About
Track-Your-Buddy is a mobile application designed for tracking friends' locations using their phone numbers, contingent on their GPS being enabled. The application enables users to pinpoint the real-time location of individuals, enhancing convenience in social meet-ups or ensuring safety by staying informed about friends' whereabouts.

## ‚ú® Features
- **Real-time Location Tracking**: Monitor friends' locations in real-time
- **Phone Number Authentication**: Secure login using phone numbers
- **Contact Integration**: Easy friend adding through phone contacts
- **Multiple Location View**: See all friends' locations on a single map
- **Privacy Controls**: GPS-dependent tracking for user privacy
- **User-friendly Interface**: Intuitive navigation and clean design
- **Location History**: Track movement patterns and history
- **Custom Markers**: Distinctive markers for different friends on map

## üîß Technical Requirements
- Android 6.0 (Marshmallow) or higher
- GPS-enabled device
- Active internet connection
- Google Play Services
- Minimum 2GB RAM
- Location permissions enabled

## üì± Usage
1. **Login/Registration**
   - Open the app
   - Enter your phone number
   - Verify with OTP
   - Grant location permissions

2. **Adding Friends**
   - Navigate to Contacts tab
   - Select friends from your contact list
   - Send tracking requests

3. **Tracking**
   - View friend locations on map
   - Click on markers for detailed info
   - Use filters to find specific friends

## üîí Privacy & Permissions
The app requires the following permissions:
- Location access (Fine and Coarse)
- Contacts read access
- Internet access
- Phone state read
- SMS read (for OTP verification)
